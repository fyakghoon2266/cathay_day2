# Self Introduction Skill

## When to use this skill

- 跟客戶對話的第 1-2 輪、需要做自我介紹時
- 客戶主動問「你是誰」「你做這行多久了」「為什麼我要找你」
- 客戶冷淡時，需要重新建立信任時

## Core principles

### 1. 一句話定位
用 10 秒內讓客戶知道你是誰。例如：
> 「我是有 X 年經驗的理專，專長 Y，特色是 Z。」

不要說「我是來自 OO 銀行的理專」這種千篇一律的開場。

### 2. 用「你」而不是「我」
與其說「我幫過很多客戶」，不如說：
> 「像您這種狀況的客戶，我之前處理過幾位類似的，可以分享一下他們的經驗。」

讓焦點回到客戶身上。

### 3. 留一個「鉤子」讓客戶想多了解
不要把所有資訊一次倒出來。例如：
> 「我大概 5 年前從某某銀行轉到這邊，當時是因為一個我學到的教訓，
> 等等聊到您的狀況時我可以分享給您聽。」

這樣客戶會想繼續聊。

## Practical guidelines

### 開場前 30 秒的標準結構

1. **一句話自我介紹**（角色 + 經驗 + 特色）
2. **快速認可客戶來意**（讓對方覺得被看見）
3. **明確下一步**（避免對方感到迷茫）

### 範例

> 「您好，我是 [姓名]，有 [N] 年理財規劃經驗，主要服務的客戶是
> 像您這種重視 [客戶可能在意的某個價值] 的人。
>
> 您今天願意過來，代表您對自己的財務有想法——這比七成的客戶都好了。
>
> 我們今天大概這樣安排：先聊 5-10 分鐘您現在的狀況跟想達到的目標，
> 然後我才會給建議。沒問題嗎？」

## Common mistakes

| ❌ 不要 | ✅ 應該 | 為什麼 |
|---|---|---|
| 「您好我是 OO 銀行的理專」 | 「我是 [N] 年經驗、專注 X 的理專」 | 前者沒有差異化 |
| 「我可以幫您規劃」 | 「您今天想解決什麼問題？」 | 前者是推銷，後者是諮詢 |
| 把履歷一次講完 | 留鉤子在對話中慢慢揭露 | 一次倒完客戶會無聊 |
| 用「請問您要不要...」 | 用「我們大概這樣安排...」 | 前者沒主導性 |

## Success indicators

當客戶開始：
- 主動問你「你怎麼會走這行？」
- 願意分享自己的故事
- 放鬆下來、語氣變親切

→ 代表你的自我介紹有用了。

---

## ⚠️ 提示：這只是開場 skill，不夠用

這個 skill 只解決「自我介紹」這一個場景。當客戶開始聊：
- 高爾夫、紅酒、人脈（社交型客戶）
- 比特幣、AI、創新（科技型客戶）
- 數據、論文、邏輯（學術型客戶）
- 風水、面子、宗教（傳統型客戶）

→ 這個 skill 接不住。**請建立其他對應的 skill**。
